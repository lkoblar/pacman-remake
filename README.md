# Pac-Man Remake

A Pac-Man remake built with Python and Pygame.

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Controls

| Key | Action |
|-----|--------|
| WASD / Arrow Keys | Move Pac-Man |
| Enter | Select / Start |
| ESC | Pause / Back to Menu |
